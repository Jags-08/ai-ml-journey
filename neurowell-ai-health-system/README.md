## 🧠 About NeuroWell

NeuroWell is a **privacy-first AI-powered health intelligence system** that runs locally and provides personalized insights by analyzing user health data over time.

It focuses on **pattern detection, smart recommendations, and proactive healthcare**, rather than just reactive symptom checking.
